새로운 태그 생성

확인 버튼 클릭하면 p태그를 생성 result1번에 추가
ptag = $("<p id='p1' onclick='proc1(this)'>Hello</p>")
  $("#result1").append(ptag);

function proc1(tt){
  $(tt).css("font-size", "+=5px");
}

새로운 태그 생성

확인 버튼 클릭하면 p태그를 생성 result2번에 추가
$(".box2 input[type=button]").on("click", function(){
  ptag = $("<p></p>", {
    'id' : 'p2',
    'text' : 'Hello',
    'click' : function(){
    $(this).css("font-size","+=5px");
    }
  });
  $("#result2").append(ptag);
})
