상태필터

폼의 요소들의 상태에 따라서 선택한다
disabled - true(사용불가능) / false(사용가능)
prop()메소드를 상태값을 가져오거나 설정한다.
$("선택자").prop("disabled") - get
$("선택자").prop("disabled", false) - set
확인버튼을 클릭하면 id의 입력요소를 사용 가능과 사용 불가능으로 번갈아 설정한다
현재 상태값을 얻어온다
현재 상태의 반대로 다시 설정한다.
